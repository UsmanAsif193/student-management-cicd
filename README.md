## Project Overview
Complete CI/CD pipeline implementation using Jenkins, Docker, Kubernetes, Prometheus, and Grafana deployed on AWS EC2.

## Application
Simple Student Management System with:
- Backend: Node.js (Express)
- Database: PostgreSQL (Dockerized)
- Features: Add, view, and manage student records

## DevOps Tools
- **Source Control**: GitHub
- **CI/CD**: Jenkins
- **Containerization**: Docker + DockerHub
- **Orchestration**: Kubernetes
- **Monitoring**: Prometheus + Grafana

## Directory Structure
```
├── app/                    # Application source code
├── docker/                 # Docker configurations
├── kubernetes/             # K8s manifests
├── jenkins/                # Jenkins pipeline
```
